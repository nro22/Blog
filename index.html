About This Blog

A personal project with a mobile-first responsive design. Created and designed by myself from scratch using vanilla HTML, CSS, and a little bit of JavaScript.

Background image for home page from Umut Bilut (Instagram handle: umitbu) via unsplash.com within licensing agreement


Possible future features and improvements include:

  • A nav bar and functioning portfolio site as the home page
  • Allow user the option to display desktop version on mobile
  • Allow user to sort posts by sequential/chronological order
  • Automate or improve structure/format to allow for new posts to appear on home page (requires backend?)
  • Allow user to modify font size via a scroll bar
  • Allow user to change theme to light/dark
  • Allow user to add comments (requires backend?)
  • Create back/forward buttons to view next/previos posts

After the creation of Posts 0-3, I implemented the use of AI via the Cursor editor with Claude 3.5 Sonnet for consulting and rapid deployment of some features. However, the vast majority of each webpage and their components were created and designed from scratch
